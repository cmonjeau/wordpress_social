## Worpdress social with docker

Install wordpress on a docker image with social plugins:

 - Buddypress
 - Buddypress docs
 - Bbpress
 - Buddydrive

 admin plugins:

 - Easy WP SMTP
 - LDAP LPRM
 - WP UserOnline

 and social theme:

 - Socialize lite

 ## Build the image

 docker build -t username/wordpress .
